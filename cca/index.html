Xiangqi.com userscript

Real-time Chinese chess assistant for xiangqi.com

CCA uses Pikafish to read the current position, draw hint arrows, show move evaluation, and support auto move. The panel and settings are embedded directly into the interface for a cleaner and faster workflow.

Core features

Hints directly on the board

Read FEN directly from xiangqi.com, analyze with the engine, and draw arrows on the live board.

Smaller panel footprint

When no board is detected, the panel collapses to its header. Once a board is found, it expands automatically.

Built-in settings

No separate popup is required. Depth, hint lines, arrow colors, auto move, and backup all live inside the panel.

Installation

1. Install Tampermonkey

If your browser does not already have a userscript manager, install Tampermonkey first.

2. Install CCA

Open the install link below and approve the installation in Tampermonkey.

https://raw.githubusercontent.com/nguyenphanvn95/nguyenphanvn95.github.io/main/cca/cca.user.js

Usage

  1. Open xiangqi.com and enter a game.
  2. CCA detects the board automatically. While the board is not ready, the panel stays collapsed.
  3. Click the gear button to open settings inside the panel.
  4. Adjust depth, hint count, arrow style, auto-move delay, and shortcut key as needed.
  5. If you need a separate board view, open the analysis board.
