Hãy import JSON từ Anki để bắt đầu

Hướng dẫn sử dụng

  1. Trong Anki, nhấn nút "Copy JSON" ở mặt sau thẻ
  2. Quay lại trang này và dán JSON vào khung Import
  3. Nhấn "Import JSON" để thêm từ vào danh sách
  4. Lặp lại với các từ khác
  5. Nhấn "Export CSV cho Anki" để tải file CSV về máy
  6. Trong Anki: File → Import → Chọn file CSV → Chọn Note Type → Tick "Allow HTML in fields" → Import

Lưu ý quan trọng:

  • Dữ liệu được lưu trong bộ nhớ trình duyệt, sẽ mất khi xóa cache
  • Khi import vào Anki, phải tick "Allow HTML in fields" để ảnh hiển thị đúng
  • Trường Image phải được mapping đúng với field chứa HTML trong Note Type của bạn
  • File CSV chỉ chứa dữ liệu từ vựng, không có dòng header
