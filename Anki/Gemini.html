🔑 Hướng dẫn lấy API Key và Model Gemini

Bước 1: Vào trang đăng ký API của Google

Truy cập: https://makersuite.google.com/app/apikey

Đăng nhập tài khoản Google (nếu được yêu cầu), sau đó nhấn nút "Create API key" để lấy mã.

Sao chép đoạn mã hiển thị, dán vào ô API Key trong phần cài đặt addon.

Bước 2: Chọn Model Gemini phù hợp

Hiện tại addon này hỗ trợ một số model như:

  • gemini-pro – dùng chung nhất
  • gemini-1.5-pro-latest – nếu bạn được cấp quyền thử nghiệm
  • gemini-2.0-flash – phiên bản rút gọn nhanh

Bạn có thể dán tên model vào ô Model Gemini trong phần cài đặt addon.

Bước 3: Kiểm tra hoạt động

Sau khi nhập đúng API và Model, hãy thử tạo 1 thẻ có trường Vocabulary. Nếu dữ liệu tự động điền, tức là kết nối đã thành công.
